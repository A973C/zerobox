zerobox
=======

Lightweight lightbox plugin for [Jquery](http://jquery.com) JavaScript library.

## Usage

HTML:

    <a class="zerobox" href="image.jpg">
        <img src="image-thumb.jpg" />
    </a>

JS:

    $('a.zerobox').zerobox();
